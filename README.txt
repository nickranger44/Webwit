WELCOME TO WEBWIT!

Webwit is currently a website cookie manipulating tool that will be turned in as a final project for CSI-160-01.

Installation steps:
1. Move the "Webwit-master directory to your desktop
2. Open up terminal and type in the following commands
3. cd ~/Desktop/Webwit-master/
4. chmod 700 install.sh
5. ./install.sh

Useage steps:
1. Open up terminal and change your directory to the location of Webwit.py
2. Type "python3 Webwit.py"
3. Enjoy!
